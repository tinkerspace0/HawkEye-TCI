Get temperature (1 pixel)

Parameters

_y: y coordinate
Remarks
This function returns temperature for given pixel. RecvImage function should be called before calling this function. Returned values are modified to fit into 2byte. To get temperature, user should subtract 5000 and then divide by 100, i.e. (real temperature value) = (return value - 5000) / 100

§ GetShutterPt100RawValue()

virtual unsigned short i3::TE_A::GetShutterPt100RawValue ( )
virtual

Get PT100 sensor output.

Returns
PT100 sensor output
Remarks
PT100 sensor is attached to shutter in order to calibrate temperature and it returns this PT100 sensor output.
PT100 output value is updated when shutter is closed. Therefore if shutter is not closed for long time, it may return incorrect value. User should close shutter periodically to get correct value by calling ShutterCalibrationOn or SetShutterMode function.
Also, TE sends this value at the end of image data, so it returns PT100 output which is received at last RecvImage function call. To get current PT100 output, RecvImage needs to be called before calling this function.

Receive image data.

Parameters
pRecvImage: variable to get image data (384x288 for QVGA and 640x480 for VGA, 16bits)
_AGC_On: if set true, AGC algorithm is applied. Otherwise, AGC is not applied.
Returns
1: Read image successfully
2: Read image of size 0
3: Recieved image size is not equal to requested size

Remarks
This function receive data from Thermal Expert and calibrate to show image. It send image data to pRecvImage. If _AGC_On is set to true, gain and offset value is automatically calculated so that image data will be fitted to 16bit range.

Receive image data with temperature limit.

Parameters
img_buf: variable to get image data (384x288 for QVGA and 640x480 for VGA, 16bits)
temp_min: minimum temperature to express in image
temp_max: maximum temperature to express in image
Returns
1: Read image successfully
2: Read image of size 0
3: Recieved image size is not equal to requested size
4: Data read fail
Remarks
This function receive image data from Thermal Expert and manipulate data to express scene within temperature range. if temperature is smaller than temp_min, 0 will be assigned, and if temperature is greater than temp_max, 65535 will be assigned. temperature within those limit will set value between 0 and 65535 depending on temperature.

§ SetShutterMode()

virtual bool i3::TE_A::SetShutterMode ( unsigned short  _mode)
virtual

Set shutter operation mode.

Parameters
_mode: Shutter operation mode. (Time mode = I3_TIME_SHUTTER, Temperature mode = I3_TEMP_SHUTTER)
Returns
true : Signal is transferred successfully
false : Transfer of signal failed
Remarks
Shutter in TE engine is automatically closed and do calibration to maintain high image quality. This function set a condition when shutter is closed. It supports time mode which closes shutter periodically with specified period and temperature mode which closes shutter when temperature of shutter is changed with specified amount.
See SetShutterTemperature and SetShutterTime.
