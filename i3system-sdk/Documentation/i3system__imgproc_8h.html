cv::Mat &  _dstImg,
double  _factor 
)

Apply AIE algorithm.

Parameters
_srcImg: Source image to apply AIE algorithm (cv::Mat with CV_8UC1)
_dstImg: AIE applied Output image (cv::Mat with CV_8UC1)
_factor: Edge enhancement factor. As the _factor value is set higher, edge will be more enhanced but image will be more noisier.
Remarks
This function applies AIE algorithm to enhance edge in _srcImg and returns output image in _dstImg. This function get 8bit gray image, so 16bit image data which is received from TE should be converted to 8bit before apply AIE.
